Техническое задание: Чат-приложение 

1. Цель проекта

Создать приложение на Pyqt с возможностью:

* создавать чаты,
* отправлять, и получать сообщения,
* менять обои для чатов
* с бесполезными функциями)

---

2. Функциональные требования

2.1 Пользователи

* Пользователи идентифицируются уникальным id и username
* Возможность получать список чатов, в которых участвует пользователь.
* Возможность регистрироваться и входить в приложение

2.2 Чаты

* Создание чата с 2 пользователями
* Получение существующего чата между двумя пользователями
* Удаление чата между двумя пользователями 
* Получение всех чатов пользователя

2.3 Сообщения

* Отправка сообщения
* Получение всех сообщений между двумя пользователями, сортировка по времени отправки 
* Работа в реальном времени с помощью WebSocket

---

3. Архитектура

* Язык: Python 3.13

* ORM:  SQLModel

* База данных: SQLite

* Модели:

  * User
  * Chat
  * Message

* для работы с базой данных используется слой Repository


5. Требования к реализации

* Все операции с БД через сессии SQLModel.
* Использовать SQLite
* Использовать все изученные технологии
